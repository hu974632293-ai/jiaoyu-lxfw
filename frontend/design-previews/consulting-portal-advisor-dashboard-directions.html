官网首页 + 顾问后台：3 套 UI 方向

这份预览只用于选择风格，不接入正式 React 路由。三套都遵守“公开官网门户 + 登录入口 + 顾问客户增长后台”的边界：官网不展示内部 CRM，顾问后台保留新建线索、研判、推荐、跟进和客户 360 入口。
