# 测试对象认领表

## 使用规则

1. 验收开始前统一初始化一次演示数据；开始后不要中途重置。
2. 本表按“列表序号”分配对象，验收人打开对应页面后把实际 ID、名称和 URL 填入记录栏。
3. 每个人只修改自己认领的对象；复核时可以查看他人对象，但不要改状态。
4. 新建对象必须使用唯一前缀：`A-验收`、`B-验收`、`C-验收`、`D-验收`、`E-验收`。
5. 如果某个对象缺失或列表排序变化，使用同类未认领对象替代，并在备注中说明。
