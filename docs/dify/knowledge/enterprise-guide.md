# 澜桥国际教育 — 企业新人指南

> 适用场景：`enterprise_guide`（企业新人指南 Agent）

---

## 1. 公司简介

澜桥国际教育是一家专注于留学规划和职业教育的服务机构，主要业务方向包括新加坡本科升学、德国双元制职业教育、亚洲多地区本科与奖学金申请，以及国际课程转轨。

核心价值观：专业、负责、陪伴。

---

## 2. 入职第一周标准流程

新人入职按以下步骤推进，由跟岗导师全程陪同：

1. **第一天**：HR 开通企业账号（系统、邮箱、企业微信），领取工位和设备，阅读员工手册。
2. **第二天**：熟悉企业微信和企业助手，了解各部门职责和联系人，完成制度学习确认。
3. **第三天至第五天**：跟岗学习，跟随导师参加客户会议或学生服务，完成首份日报。
4. **第一周结束**：提交首周学习总结，与导师做一对一反馈。

## 4. 日报和周报制度

- **日报**：每个工作日提交，内容包括当日完成事项、风险点和次日计划。通过企业助手"日报"入口提交。
- **周报**：每周一自动汇总上周所有员工日报，管理者可查看团队周报。
- **风险处理**：日报中标注的风险（如客户材料不齐、签证延迟等），由管理者判断是否需要协调处理。

## 5. 常见制度速查

| 问题 | 答案 |
|------|------|
| 请假怎么申请？ | 在企业助手中提交申请，由部门负责人审批。 |
| 出差报销流程？ | 提交出差申请 → 保留发票 → 回公司后提交报销单 → 财务审核。 |
| 客户资料如何录入？ | 使用企业助手"客户录入"功能，或通过顾问工作台的"新增客户"入口。 |
| 公司文件在哪里找？ | 企业微信微盘或企业助手的知识库。 |

## 7. 新人常见问题

**Q: 遇到不懂的问题应该问谁？**
A: 优先问你的跟岗导师。如果是跨部门问题，可以在企业微信上联系对应部门的负责人。

**Q: 客户问我不会回答的问题怎么办？**
A: 诚实地告诉客户"我需要确认后给您回复"，然后请教导师或升学规划部同事。

**Q: 如何使用企业助手？**
A: 在员工工作台可以看到"企业助手"入口，支持对话方式查询组织架构、提交日报草稿等。
